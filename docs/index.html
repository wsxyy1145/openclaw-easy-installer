🚀 Installation Modes

Standard Mode

Install from official npm registry. Best for international users.

npm install -g openclaw@latest

Speed: Normal | Network: Required

RECOMMENDED
China Mirror

Faster downloads with Chinese CDN mirror. Perfect for mainland China users.

PowerShell Script + npm

Speed: Fast ⚡ | Network: Required

Offline Mode

Install from local package. No internet connection needed.

npm install -g openclaw-offline.tgz

Speed: Fastest | Network: Not Required

Development

Install latest development version with newest features.

npm install -g openclaw@dev

Speed: Fast | Stability: Beta

📦 Quick Start

1

Download

Download the latest release from GitHub Releases

2

Run as Administrator

Right-click Install-OpenClaw.bat and select "Run as administrator"

3

Select Mode

Choose your preferred installation mode (China Mirror recommended for faster downloads)

4

Complete

Wait for installation to complete and access control panel at http://localhost:18789
