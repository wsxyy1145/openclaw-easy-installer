### 🚀 使用方法

#### GUI安装（推荐）
```bash
# 直接运行批处理文件
Install-OpenClaw.bat

# 或者运行Python脚本
python install_gui.py
```

#### 命令行安装
```bash
# 标准安装
python install.py

# 使用中国镜像
python install.py --install-mode cn-mirror

# 离线安装
python install.py --install-mode offline

# 开发版安装
python install.py --install-mode dev

# 不创建快捷方式
python install.py --no-shortcuts

# 查看帮助
python install.py --help
```

---

### 📝 注意事项

1. **管理员权限**: 必须以管理员身份运行
2. **网络要求**: 除离线模式外需要网络连接
3. **磁盘空间**: 确保至少2GB可用空间
4. **杀毒软件**: 某些杀毒软件可能误报，请添加例外
5. **防火墙**: 允许Node.js和npm通过防火墙

### 🔧 故障排除

#### 问题1: 依赖项安装失败
**解决方案**:
- 检查网络连接
- 尝试使用中国镜像模式
- 手动安装依赖: `pip install requests pypiwin32`

#### 问题2: Node.js安装失败
**解决方案**:
- 手动下载安装: https://nodejs.org/
- 检查Windows Installer服务是否运行
- 重启计算机后重试

#### 问题3: OpenClaw安装超时
**解决方案**:
- 使用中国镜像模式: `--install-mode cn-mirror`
- 检查npm代理设置
- 尝试离线安装模式

#### 问题4: 权限不足
**解决方案**:
- 右键点击 -> "以管理员身份运行"
- 检查用户账户控制(UAC)设置
